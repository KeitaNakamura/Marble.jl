Poingr.ShapeValues — Method
Poingr.ShapeValues(::ShapeFunction)
Poingr.ShapeValues(::Type{T}, ::ShapeFunction)

Construct object storing value of ShapeFunction.

Examples

julia> sv = Poingr.ShapeValues(QuadraticBSpline{2}());

julia> update!(sv, Grid(0:3, 0:3), Vec(1, 1));

julia> sum(sv.N)

julia> sum(sv.∇N)

Poingr.update! — Function
update!(::ShapeValues, grid::Grid, x::Vec)
update!(::ShapeValues, grid::Grid, indices::AbstractArray, x::Vec)

Update value of shape function at x with each grid node.

Examples

Poingr.BSplinePosition — Type
Poingr.BSplinePosition(; nth::Int, dir::Int)

Position of node at ξ = 0. The node is located at nth away from bound with dir direction.

Examples

julia> Poingr.BSplinePosition([1,2,3,4,5], 2)
BSplinePosition(nth = 1, dir = 1)

julia> Poingr.BSplinePosition([1,2,3,4,5], 4)
BSplinePosition(nth = 1, dir = -1)

julia> pos = Poingr.BSplinePosition([1,2,3,4,5], 1)
BSplinePosition(nth = 0, dir = 0)

julia> f = QuadraticBSpline{0}()
QuadraticBSpline{0}()

julia> f(0.0, pos)
1.0
